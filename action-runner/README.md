## 사용방법

### 0. GITHUB_TOKEN 환경변수 설정
* github 계정의 token을 GITHUB_TOKEN=$value 환경변수로 설정
* 동적으로 action runner token 값을 가져오기 위함

### 1. Install & Start
* token 값은 runner 생성 페이지에서 확인
```bash
setup-runner.sh
```

### 2. Stop
```bash
stop-runner.sh
```

### 3. Start
```bash
start-runner.sh
```

### 4. Remove
```bash
remove-runner.sh
```
